Description of Project


Upload the file using the selection functionality or use the existing uploaded data set (Iris data set Or Wine data set)

Once the data set is selected, select the column you want as the target column, select color schema and select the height and width of the application to determine the radius

Project structure and libraries

Libraries used are: bootstrap, jquery and d3. D3 is the only library used for plotting

Data is uploaded in the data folder

Scripts are uploaded in the src folder

Index.php has been added to deploy it to Heroku, it serves no other purpose
